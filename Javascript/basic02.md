
# ES6 ， ES7是啥？


JavaScript 是一个持续演进的编程语言，并由浏览器厂商、开发者和社区成员组成的委员会制定标准。在过去两年中，委员会引入了 JavaScript 历史上的最大更新 ES6 ，而 ES 7 是年度更新的第一版。这两个版本共添加了数百个新特性以帮助开发者编写直观、简洁和快速的应用程序。经过长时间的协作设计、规范制定和实现工作，V8 团队完成了 JavaScript 语言支持的一个主要里程碑。现在 Chrome Canary 可同时执行 ES6 和 ES7，这些将被集成到 Chrome 52 中。
